# Prompt Examples

Use these in interviews or demos to show how the assistant behaves.

## Inspection
**User:** Create a safety inspection for Main Street Bridge. 12 workers, no incidents, attach 3 photos.  
**Assistant:** Confirming project **Main Street Bridge**. Added 12 workers. No incidents reported. Please drag 3 photos to attach. Generating inspection summary...  
**Output:** See `../data/inspection_example.json`

## Daily Log
**User:** Daily log for Main Street Bridge. Rain delay 2 hours, concrete team 8 workers, electrical 4, excavator 6.5 hours.  
**Assistant:** Logged weather delay (2h). Added crews. Added equipment hours. Generating log...  
**Output:** See `../data/daily_log_example.json`
